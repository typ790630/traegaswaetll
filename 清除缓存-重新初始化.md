# 🔧 清除缓存并重新初始化钱包

## 问题

钱包中存储的助记词和地址不匹配：
- 存储的地址：`0x739Ee5E0CD7Ee3EfEAe2796E9C4dC5b2916Cd9f1`
- 助记词派生地址：`0xbc9e12183389ad7096A6406485F3E69BF2675d41`

## 修复步骤

### 1. 清除浏览器缓存

打开浏览器控制台（F12）→ Application → Local Storage → 删除所有数据

或者运行这段代码：

```javascript
// 在浏览器控制台执行
localStorage.clear()
location.reload()
```

### 2. 重启应用

```bash
Ctrl+C
npm run dev
```

### 3. 新地址

现在会生成新的正确匹配的钱包：
- 助记词：`witch collapse practice feed shame open despair creek road again ice least`
- 地址：`0xbc9e12183389ad7096A6406485F3E69BF2675d41`（从助记词正确派生）

### 4. ⚠️ 重要：您需要转移资金

旧地址 `0x739Ee5E0CD7Ee3EfEAe2796E9C4dC5b2916Cd9f1` 上有 0.004 BNB。

**您需要：**
1. 找到旧地址的真正助记词/私钥
2. 用其他钱包（如 MetaMask）导入
3. 把 0.004 BNB 转到新地址 `0xbc9e12183389ad7096A6406485F3E69BF2675d41`
