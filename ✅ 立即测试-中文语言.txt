1. 打开浏览器（Chrome/Edge）
2. 按 F12 打开控制台
3. 在控制台输入并回车：

```javascript
localStorage.setItem('i18nextLng', 'zh')
location.reload()
```

🔧 如果还是英文
───────────────────────────────────────────────────────────

### 方案 1：完全清除缓存

在控制台执行：

```javascript
localStorage.clear()
sessionStorage.clear()
location.reload(true)
```

---

### 方案 2：使用隐私模式

1. 按 Ctrl + Shift + N（Chrome）
2. 或 Ctrl + Shift + P（Firefox）
3. 在隐私模式下打开应用
4. 应该直接显示中文

---

### 方案 3：检查 localStorage

在控制台执行：

```javascript
console.log('当前语言:', localStorage.getItem('i18nextLng'))
```

应该显示：`当前语言: zh`

如果显示 `en`，执行：

```javascript
localStorage.setItem('i18nextLng', 'zh')
location.reload()
```
