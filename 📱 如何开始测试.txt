🛠️ 高级选项：性能监控脚本
───────────────────────────────────────────────────────────

如果您熟悉浏览器控制台，可以使用性能监控脚本：

### 步骤：

1. **打开浏览器控制台**
   - Chrome：菜单 → 更多工具 → 开发者工具 → Console
   - Safari：设置 → 高级 → 显示开发菜单 → 开发 → 显示 JavaScript 控制台

2. **加载监控脚本**
   - 打开 "performance-monitor.js"
   - 复制所有代码
   - 粘贴到控制台
   - 按回车执行

3. **执行测试**
   - 正常进行兑换和转账测试
   - 脚本会自动记录所有性能数据

4. **查看报告**
   - 在控制台输入：perfMonitor.getReport()
   - 按回车查看详细性能报告

5. **导出数据**
   - 在控制台输入：perfMonitor.export()
   - 自动下载性能数据 JSON 文件

---

**提示：** 这是可选的高级功能，不影响基本测试
